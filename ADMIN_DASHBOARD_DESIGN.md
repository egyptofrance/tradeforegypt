# تصميم لوحة التحكم المتقدمة - Trade for Egypt

## المتطلبات الوظيفية

### 1. Dashboard الرئيسي
- **إحصائيات حية:**
  - عدد العائلات
  - عدد الماركات
  - عدد المنتجات
  - عدد الصفحات المولدة
  - عدد الصفحات المتوقعة
  - نسبة الإنجاز (Progress %)
- **رسوم بيانية:**
  - توزيع الماركات حسب العائلات
  - الصفحات المولدة vs المتوقعة
- **روابط سريعة:**
  - إضافة ماركة جديدة
  - عرض المواقع
  - معاينة الصفحات

### 2. صفحة المواقع (Sites)
- **عرض جميع السبدومينات:**
  - اسم الماركة
  - السبدومين (zanussi.tradeforegypt.com)
  - عدد الصفحات
  - حالة التوليد (Progress Bar)
  - زر "عرض الصفحات"
  - زر "معاينة الموقع"

### 3. صفحة الصفحات (Pages)
- **عرض جميع الصفحات لماركة معينة:**
  - المسار (URL)
  - الكلمة البحثية
  - المنتج
  - حالة التوليد (مولدة / قيد الانتظار)
  - زر "معاينة"
  - زر "تحرير المحتوى"

### 4. نظام المعاينة (Preview)
- **معاينة الصفحة قبل النشر:**
  - عرض كامل للصفحة في iframe
  - إظهار Meta Tags
  - إظهار Schema Markup
  - زر "نشر الآن"

### 5. Progress Bar للتوليد
- **شريط تقدم حي:**
  - عدد الصفحات المولدة / المتوقعة
  - نسبة الإنجاز
  - الوقت المتبقي المتوقع
  - زر "بدء التوليد"
  - زر "إيقاف"

## البنية التقنية

### الصفحات المطلوبة:
1. `/admin` - Dashboard الرئيسي
2. `/admin/sites` - عرض المواقع
3. `/admin/sites/[brand]` - عرض صفحات ماركة معينة
4. `/admin/preview/[brand]/[product]/[keyword]` - معاينة صفحة
5. `/admin/generate` - صفحة التوليد مع Progress Bar

### API Routes المطلوبة:
1. `/api/admin/stats` - جلب الإحصائيات
2. `/api/admin/sites` - جلب جميع المواقع
3. `/api/admin/pages` - جلب صفحات ماركة
4. `/api/admin/generate-pages` - بدء توليد الصفحات
5. `/api/admin/generate-status` - حالة التوليد
